# Dental Hygiene Test Bank

A React-based test bank application for dental hygiene students.

## Deployment Location

This application is configured to run at:
```
https://m2eacircle.com/education/testbank/
```

## Quick Start

### 1. Install Dependencies
```bash
npm install
```

### 2. Build for Production
```bash
npm run build
```

### 3. Deploy
Upload the contents of the `build/` folder to your server at `/education/testbank/`

## Configuration

The application is pre-configured for deployment at `/education/testbank/`:

- `package.json` → `"homepage": "/education/testbank"`
- `.env` → `PUBLIC_URL=/education/testbank`

## Development

```bash
npm start
```

Runs the app at http://localhost:3000

## Features

- Multiple lessons and subjects
- Practice mode with immediate feedback
- Timed test mode
- Performance tracking
- Mobile responsive design

## Technologies

- React 18.2.0
- Tailwind CSS
- Lucide React Icons
